# Real-Time-Face-Detection-with-Python-OpenCV
Real-time face detection system developed with Python and OpenCV using Haar Cascade classifiers.
This repository contains a Computer Vision project developed in Python. It implements face detection using Haar Cascade classifiers, optimized for real-time performance through grayscale frame processing.

# Key Features
* Instant face detection via webcam.
* Efficient image processing to ensure low latency.
* Professional project structure with isolated dependencies.

# Tech Stack
* Python 3.12+
* Library OpenCV
* Virtual Environment (venv)
* Visual Studio Code

# Activate the environment:
Open your VS Code Terminal and run:
* Bash:
.\venv\Scripts\activate

# Install required packages:
* Bash:
pip install -r requirements.txt

# Run the application:
* Bash: 
python Main.py

# Project Structure

    Main.py → Core script for video capture and detection.

    Models → Directory for pre-trained XML classifiers.

    requirements.txt → Project dependencies list.

    .gitignore → Excludes venv/ and cache files from the repository.
